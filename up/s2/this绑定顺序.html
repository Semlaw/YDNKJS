1.默认绑定：
			独立函数调用，可以看做是无法应用其他规则时的默认规则
			eg：
			function foo(){
				console.log(this);
			}
			foo();
			在非严格模式下this指向全局对象；
		
			2.隐式绑定
			eg：
			function foo(){
				console.log('this.a');
			}
			var obj = {
				a:2,
				foo:foo
			}
			obj.foo()//2
		
			3.显式绑定
			通过call或apply方法调用函数
			
			显示绑定 硬绑定
			bind方法
		
			判断this
			1.函数是否在new中调用(new 绑定)？如果是的话this绑定的是新创建的对象
			var bar  = new foo()
			2.函数是否通过call、apply（显式绑定）或者硬绑定（bind）？如果是的话this绑定是新创建的对象。
			
			var bar = foo.call(obj2)
			3.函数是否在某个上下文对象中调用（隐式绑定）？如果是的话，this绑定的是哪个上下文对象。
			var bar = obj1.foo()
			4.如果都不是的话，使用默认绑定，如果在严格模式下就会绑定到undefined，否则会绑定到全局对象上。
			var bar = foo()
		
			在非严格模式下使用call、apply、bind进行显式绑定、并使用undefined或null作为参数时，this会指向全局对象
